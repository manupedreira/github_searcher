# github_searcher

Simple app developed in React Native that fetches and show the repos of any given Github member

# Running the app

- Install dependencies: npm install
- Launch the app: react-native run-ios | react-native run-android

# Definition

The app is based on three screens:

- Searcher: Allows to enter a Github user.
- Error: Shows an error when the given user doesn't exist or has no repos.
- Repos: Shows the repos from the given user.
